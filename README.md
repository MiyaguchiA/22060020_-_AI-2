#　犬の画像アプリ

## アプリ概要
Streamlitを使った犬の画像表示アプリで、  
ユーザーが犬種を選択すると、その犬種のランダムな画像を表示します。  
また、ランダムに犬の画像を表示するボタンもあります。
 s
## 使用API
- [Dog CEO's Dog API](https://dog.ceo/dog-api/)

## ファイル構成
- `app.py`: ボディ的なユーザー操作部分
- `dog_api.py`: Dog CEOのAPIを呼び出す部分
